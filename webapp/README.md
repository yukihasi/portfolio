# pikopiko
## ピアノの鍵盤をベースに作曲するサイト(主に8bit音源)一部未完成

# 完成リスト
## 基本操作
マウスによるクリック

キーボードによる入力で音が鳴る

## ログイン画面
・ピアノの鍵盤

・ログイン機能

・新規ユーザー登録機能

## ログイン後の画面(ホーム画面)
### 左端のメニュー
作曲に必要な機能一覧

・楽器：楽器の選択

・録音：タイトルを決めて録音。録音した作曲データはデータベース(MySQL)に保存される

・ノイズ：サンプリングレート、インターバルタイム、タイムを指定することで様々なノイズを再現
### 画面右上のメニュー
ユーザー管理メニュー

・Logout：ログアウト機能
